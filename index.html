Instructions

How to play this game?

  1. Press 'A' to Start.
  2. The game will give the first signal. Repeat the signal by pressing the same color lens.
  3. The game will duplicate the first signal and add one. Repeat these two signals by pressing the same color lenses, in order.
  4. The game will duplicate these first two signals and add one.
  5. Repeat the signal by pressing the same color boxes.
  6. Continue playing as long as you can repeat each sequence of sig- nals correctly.
  7. If you fail to repeat a sequence exactly, The game will respond you with a "RAZZ" sound. This means you've lost the game.
